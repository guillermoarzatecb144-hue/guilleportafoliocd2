Componentes de Arduino

Sensores, actuadores y hardware compatible

¿Qué es Arduino?

Arduino es una plataforma de desarrollo de hardware y software de código abierto que permite crear proyectos electrónicos interactivos. Su núcleo es un microcontrolador (generalmente ATmega) con pines de entrada y salida que se pueden conectar a sensores, actuadores y otros módulos.

El entorno de programación usa un lenguaje basado en C/C++ con dos funciones principales: setup() que se ejecuta una sola vez al iniciar, y loop() que corre indefinidamente.

Sensores

Los sensores capturan información del mundo físico y la convierten en señales eléctricas que el microcontrolador puede leer.

📡
HC-SR04
ULTRASONIDO
Mide distancias de 2 cm a 4 m usando pulsos de ultrasonido. Ideal para detección de obstáculos y robots.
🌡️
DHT11 / DHT22
TEMP & HUMEDAD
Mide temperatura (0-50°C) y humedad relativa (20-80%). DHT22 es más preciso y con mayor rango.
☀️
LDR (Fotoresistor)
LUZ AMBIENTAL
Resistencia que varía según la intensidad de luz. Útil para encendido automático y automatización lumínica.
📳
MPU-6050
GIROSCOPIO / ACELERÓMETRO
Mide aceleración y velocidad angular en 3 ejes por I2C. Usado en drones, robots de equilibrio y wearables.
🔥
MQ-2 / MQ-135
GAS / CALIDAD AIRE
Detecta gases como humo, CO, CO₂ y vapores inflamables. Muy usado en sistemas de alarma y seguridad.
💧
YL-83 / FC-28
HUMEDAD DE SUELO
Detecta el nivel de humedad en tierra. Fundamental para sistemas de riego automático agrícola.
🔍
PIR HC-SR501
MOVIMIENTO / PIR
Detecta movimiento de personas y animales mediante radiación infrarroja. Rango típico: 7 m, ángulo 120°.
🔊
KY-037 / KY-038
SONIDO / MICRÓFONO
Detecta niveles de sonido. KY-037 ofrece salida analógica para medir la intensidad del sonido.

Ejemplo — Sensor ultrasonido HC-SR04

// Pines del sensor
const int TRIG = 9;
const int ECHO = 10;

void setup() {
  Serial.begin(9600);
  pinMode(TRIG, OUTPUT);
  pinMode(ECHO, INPUT);
}

void loop() {
  // Enviar pulso de 10 µs
  digitalWrite(TRIG, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG, LOW);

  // Medir duración del eco
  long duracion = pulseIn(ECHO, HIGH);

  // Calcular distancia en cm
  float distancia = duracion * 0.034 / 2;

  Serial.print("Distancia: ");
  Serial.print(distancia);
  Serial.println(" cm");
  delay(500);
}
