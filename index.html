El padre del internet

Cultura Digital 2  ·  Inteligencia Artificial

🌐

Vint Cerf & Bob Kahn

Considerados los padres del internet, crearon en 1974 el protocolo TCP/IP, la base técnica sobre la que opera toda la Internet moderna. Su trabajo transformó redes militares aisladas en una red global de comunicación abierta.

¿Quiénes son?

Vinton Gray "Vint" Cerf (nacido en 1943, Nuevo Haven, EE. UU.) es informático y uno de los pioneros de Internet. Trabajó en DARPA y co-diseñó el protocolo TCP/IP junto a Bob Kahn. Actualmente es vicepresidente de Google y "Chief Internet Evangelist".

Robert Elliot "Bob" Kahn (nacido en 1938, Brooklyn, EE. UU.) es ingeniero eléctrico. Fue quien originalmente diseñó ARPANET, la red precursora de Internet, y junto a Cerf publicó el histórico artículo de TCP/IP en 1974.

Historia del internet

La Internet no fue inventada por una sola persona, sino que es el resultado de décadas de colaboración científica. Todo comenzó durante la Guerra Fría, cuando el gobierno de los Estados Unidos buscaba una red de comunicación que pudiera sobrevivir un ataque nuclear.

En 1969, la ARPA (Agencia de Investigación Avanzada de Defensa) creó ARPANET, la primera red de computadoras que conectó cuatro universidades estadounidenses. El primer mensaje enviado fue "LO" (intento de escribir "LOGIN" que colapsó el sistema).

Para 1974, Cerf y Kahn publicaron el protocolo TCP/IP, que permitía que distintas redes se comunicaran entre sí de forma estándar. Este fue el momento fundacional de lo que hoy llamamos Internet.

Línea del tiempo

1958

EE. UU. crea DARPA (Agencia de Proyectos de Investigación Avanzada de Defensa).

1969

Nace ARPANET. Se envía el primer mensaje entre UCLA y Stanford.

1972

Bob Kahn organiza la primera demostración pública de ARPANET con 40 máquinas conectadas.

1974

Cerf y Kahn publican "A Protocol for Packet Network Interconnection": nace TCP/IP.

1983

ARPANET adopta TCP/IP oficialmente. Este momento se considera el "nacimiento real" de Internet.

1991

Tim Berners-Lee crea la World Wide Web (WWW), haciendo Internet accesible al público general.

2004

Cerf y Kahn reciben la Medalla Presidencial de la Libertad y el Premio Turing de la ACM por su trabajo.

¿Por qué es importante hoy?

El protocolo TCP/IP sigue siendo la base de toda comunicación en Internet hoy en día. Cada vez que abres una página web, envías un correo o usas una app, tu dispositivo usa TCP/IP para fragmentar, enviar y reensamblar los datos.

Sin el trabajo de Cerf y Kahn, no existiría el comercio electrónico, las redes sociales, el streaming de video, las videollamadas, ni la Inteligencia Artificial conectada a la nube que estudias en esta especialidad.

Su legado demuestra que una idea técnica bien fundamentada puede cambiar la humanidad entera.

Lenguajes de diseño y programación

Cultura Digital 2  ·  Inteligencia Artificial

¿Qué es un lenguaje de programación?

Un lenguaje de programación es un conjunto formal de reglas, símbolos e instrucciones que permiten a los humanos comunicarse con una computadora para indicarle qué tareas debe realizar.

Existen cientos de lenguajes, cada uno diseñado con un propósito específico: diseño web, desarrollo de aplicaciones, análisis de datos, inteligencia artificial, videojuegos, entre otros.

Los lenguajes se clasifican en lenguajes de alto nivel (cercanos al lenguaje humano, como Python o JavaScript) y bajo nivel (más cercanos al hardware, como Assembly).

Categorías de lenguajes

MARCADO / MARKUP ESTILOS / STYLE SCRIPTING / FRONTEND BACKEND / SERVIDOR DATOS / DATA SCIENCE

Los lenguajes de marcado describen la estructura del contenido (HTML, XML). Los de estilos controlan la apariencia visual (CSS, SASS). Los de scripting agregan interactividad (JavaScript, TypeScript). Los de backend procesan la lógica del servidor (Python, PHP, Java, Node.js). Y los de datos analizan y modelan información (Python, R, SQL).

Principales lenguajes

🐍

Lenguaje versátil y legible, favorito para inteligencia artificial, machine learning y automatización.

☕

Lenguaje robusto y multiplataforma usado en aplicaciones empresariales y desarrollo Android.

🐘

Backend · Web Server

Lenguaje de servidor muy usado en sitios dinámicos. Potencia plataformas como WordPress.

📊

Datos · Bases de datos

Lenguaje estándar para consultar, insertar y manipular datos en bases de datos relacionales.

📈

Especializado en análisis estadístico, visualización de datos y modelado matemático.

Guía rápida de HTML5

Cultura Digital 2  ·  Inteligencia Artificial

¿Qué es HTML5?

HTML5 (HyperText Markup Language versión 5) es el lenguaje estándar para crear y estructurar el contenido de páginas web. No es un lenguaje de programación, sino un lenguaje de marcado: usa etiquetas (tags) para describir qué es cada elemento del contenido.

Fue desarrollado por el W3C (World Wide Web Consortium) y el WHATWG, y se convirtió en estándar oficial en 2014. HTML5 introdujo elementos semánticos, soporte nativo para audio/video y mejoras para aplicaciones web.

Estructura básica de un documento HTML5

<!-- Declaración del tipo de documento -->
<!DOCTYPE html>

<!-- Elemento raíz -->
<html lang="es">

  <!-- Cabecera (no visible en pantalla) -->
  <head>
    <meta charset="UTF-8"/>
    <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
    <title>Mi Página</title>
    <link rel="stylesheet" href="estilos.css"/>
  </head>

  <!-- Cuerpo (contenido visible) -->
  <body>
    <h1>Hola, mundo</h1>
    <p>Mi primer párrafo.</p>
  </body>

</html>

Etiquetas principales

<h1> … <h6>Encabezados de nivel 1 al 6. h1 es el más importante.
<p>Párrafo de texto.
<a href="">Enlace/hipervínculo. href indica la URL destino.
<img src="" alt="">Inserta una imagen. alt describe la imagen (accesibilidad).
<ul> / <ol>Lista desordenada (viñetas) u ordenada (números).
<li>Elemento de lista. Siempre va dentro de ul u ol.
<div>Contenedor genérico en bloque para agrupar elementos.
<span>Contenedor en línea para aplicar estilos a fragmentos de texto.
<table>Define una tabla. Contiene tr, th, td.
<form>Formulario para recopilar datos del usuario.
<input>Campo de entrada de datos. type define su comportamiento.
<button>Botón interactivo.

Etiquetas semánticas de HTML5

HTML5 introdujo etiquetas semánticas que describen el significado del contenido, no solo su apariencia. Esto mejora la accesibilidad, el SEO y la organización del código.

Guía rápida de CSS3

Cultura Digital 2  ·  Inteligencia Artificial

¿Qué es CSS3?

CSS3 (Cascading Style Sheets versión 3) es el lenguaje utilizado para controlar la presentación visual de las páginas HTML. Mientras HTML define la estructura, CSS define cómo se ve: colores, tipografías, espaciado, layouts y animaciones.

CSS3 introdujo módulos poderosos como Flexbox, CSS Grid, variables CSS, media queries y animaciones que revolucionaron el diseño web moderno.

CSS se "cascada" significa que las reglas se aplican en orden de especificidad: estilos del navegador → estilos del autor → estilos en línea.

Sintaxis básica

/* Estructura: selector { propiedad: valor; } */

h1 {
  color: #f5c518;
  font-size: 2rem;
  font-family: 'Orbitron', sans-serif;
  text-align: center;
}

.mi-clase {           /* selector por clase */
  background-color: #222;
  padding: 20px;
  border-radius: 8px;
}

#mi-id {              /* selector por id */
  width: 100%;
  max-width: 800px;
}

El modelo de caja (Box Model)

Todo elemento HTML es una caja rectangular compuesta por capas concéntricas:

/* Desde adentro hacia afuera: */

.caja {
  width:   200px;       /* ancho del CONTENIDO */
  height:  100px;       /* alto del CONTENIDO */
  padding: 20px;        /* espacio interior */
  border:  2px solid gold; /* borde */
  margin:  16px;        /* espacio exterior */
  box-sizing: border-box; /* padding incluido en width */
}

Flexbox — Layout flexible

Flexbox permite distribuir elementos en una fila o columna de forma flexible y alineada.

.contenedor {
  display:         flex;
  flex-direction:  row;           /* row | column */
  justify-content:space-between;  /* alineación eje principal */
  align-items:     center;        /* alineación eje secundario */
  gap:             16px;          /* espacio entre items */
  flex-wrap:       wrap;          /* permite salto de línea */
}

Animaciones y transiciones

/* Transición suave al pasar el ratón */
.btn {
  background:   #f5c518;
  transition:  background 0.3s ease, transform 0.2s;
}
.btn:hover {
  background:  #fff176;
  transform:  scale(1.05);
}

/* Animación con @keyframes */
@keyframes pulsar {
  0%   { opacity: 1;   transform: scale(1); }
  50%  { opacity: 0.5; transform: scale(1.1); }
  100% { opacity: 1;   transform: scale(1); }
}
.icono {
  animation: pulsar 2s infinite ease-in-out;
}

Media Queries — Diseño responsivo

Las media queries permiten aplicar estilos diferentes según el tamaño de pantalla, haciendo el diseño responsivo.

/* Mobile first: estilos base para móvil */
.contenedor {
  width: 100%;
  padding: 16px;
}

/* Tablet (768px o más) */
@media (min-width: 768px) {
  .contenedor {
    max-width: 720px;
    margin: 0 auto;
  }
}

/* Desktop (1200px o más) */
@media (min-width: 1200px) {
  .contenedor {
    max-width: 1100px;
  }
}

Conceptos Generales de Python

Cultura Digital 2  ·  Inteligencia Artificial

¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y multiparadigma creado por Guido van Rossum y publicado en 1991. Su filosofía hace hincapié en la legibilidad del código usando una sintaxis limpia y expresiva.

Es uno de los lenguajes más populares del mundo gracias a su versatilidad: se usa en ciencia de datos, inteligencia artificial, desarrollo web, automatización, ciberseguridad y mucho más.

Características principales

📖
LEGIBLE
Sintaxis clara, parecida al inglés natural. Fácil de leer y mantener.
🔄
INTERPRETADO
No necesita compilación previa; el intérprete ejecuta el código línea por línea.
🌐
MULTIPLATAFORMA
Corre en Windows, macOS, Linux sin modificar el código.
📦
LIBRERÍAS
Vasta colección de módulos: NumPy, Pandas, TensorFlow, Flask, Django…
🤝
OPEN SOURCE
Gratuito y con una enorme comunidad activa worldwide.
🧩
MULTIPARADIGMA
Soporta programación orientada a objetos, funcional e imperativa.

Variables y tipos de datos

En Python no se declara el tipo de una variable; el intérprete lo infiere automáticamente (tipado dinámico).

# Tipos de datos básicos
nombre   = "Python"       # str  — cadena de texto
version  = 3.12           # float — número decimal
año      = 1991           # int  — número entero
activo   = True           # bool — booleano
nulo     = None           # NoneType

# Estructuras de datos
lista   = ["a", 1, True] # list  — mutable, ordenada
tupla   = (10, 20, 30)   # tuple — inmutable
dicts   = {"clave": "valor"} # dict  — clave-valor
conjunt = {1, 2, 3}      # set   — sin duplicados

# type() muestra el tipo
print(type(nombre))       # <class 'str'>

Estructuras de control

# if / elif / else
nota = 85
if nota >= 90:
    print("Excelente")
elif nota >= 70:
    print("Aprobado")
else:
    print("Reprobado")

# for — itera sobre secuencias
for i in range(5):
    print(i)              # 0 1 2 3 4

for fruta in ["manzana", "pera"]:
    print(fruta)

# while — mientras condición sea True
contador = 0
while contador < 3:
    print(contador)
    contador += 1

# break / continue
for n in range(10):
    if n == 5: break    # termina el bucle
    if n % 2 == 0: continue # salta pares
    print(n)

Funciones

Las funciones se definen con def. Python soporta argumentos por defecto, *args y **kwargs para flexibilidad.

# Función básica
def saludar(nombre = "Mundo"):
    return f"¡Hola, {nombre}!"

print(saludar())           # ¡Hola, Mundo!
print(saludar("Python"))  # ¡Hola, Python!

# *args — número variable de args
def sumar(*nums):
    return sum(nums)

print(sumar(1, 2, 3, 4)) # 10

# Función lambda (anónima)
cuadrado = lambda x: x ** 2
print(cuadrado(5))        # 25

Programación Orientada a Objetos (POO)

Python soporta clases, herencia, encapsulamiento y polimorfismo de manera sencilla.

class Animal:
    def __init__(self, nombre):
        self.nombre = nombre

    def hablar(self):
        return "..."

class Perro(Animal):      # hereda de Animal
    def hablar(self):       # sobreescribe el método
        return f"{self.nombre} dice: ¡Guau!"

rex = Perro("Rex")
print(rex.hablar())        # Rex dice: ¡Guau!
print(isinstance(rex, Animal)) # True

Componentes de Arduino

Sensores, actuadores y hardware compatible

¿Qué es Arduino?

Arduino es una plataforma de desarrollo de hardware y software de código abierto que permite crear proyectos electrónicos interactivos. Su núcleo es un microcontrolador (generalmente ATmega) con pines de entrada y salida que se pueden conectar a sensores, actuadores y otros módulos.

El entorno de programación usa un lenguaje basado en C/C++ y ofrece una estructura sencilla con dos funciones principales: setup() que se ejecuta una sola vez, y loop() que corre indefinidamente.

Ejemplo de código — LED parpadeante

// Estructura básica de un sketch de Arduino
const int LED_PIN = 13;   // pin digital 13

void setup() {
  pinMode(LED_PIN, OUTPUT); // configurar como salida
  Serial.begin(9600);       // iniciar comunicación serial
}

void loop() {
  digitalWrite(LED_PIN, HIGH); // LED encendido
  delay(1000);                 // esperar 1 segundo
  digitalWrite(LED_PIN, LOW);  // LED apagado
  delay(1000);                 // esperar 1 segundo
  Serial.println("Parpadeo!"); // imprimir en monitor
}
